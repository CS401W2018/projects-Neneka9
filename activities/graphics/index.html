How do you resize your photo to the size it should be displayed?

On mac, you can resize the photos by opening them with preview, and clicking adjust size from the tools tab. On windows, paint and photo can resize the photos.

Why should you resize photos using photo editing software instead of resizing it using CSS?

Resizing a photo with using CSS might have extra cost and it needs more battery to adjust every time.
