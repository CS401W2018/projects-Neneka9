ASCII and Unicode

ASCII stands for American Standard Code for Information Interchange. Mostly, it is for English alphabets.
Unicode is for almost all language.

What is UTF-8?

It stands for Unicode Transformation Format -8-bit.

What is URL?

It stands for Uniform Resource Locator. It is used to address to certain pages on the internet.

What is domain name?

It represents the organization that host a website. Everyone can have it when they create a website.

What is HTTP?

It stands for Hypertext Transfer Protocol. It is used to send information among electric devices.

What is HTTPS?

It stands for Hypertext Transfer Protocol Secure. It is important when people send data that needs to be protected.

What is IP Address?

It stands for Internet Protocol address. It is a unique number to include the location of the information so that others can find it from their devices.

What is DNS

It stands for Domain Name System. People can find domain names in DNS.

What is W3C Recommendations

W3C stands for World Wide Web Consortium. It is a guideline that explains how the technologies work.

What is XML, XHTML, HTML5

All of them are markup languages for the web. XML stands for eXtensible Markup Language. It stores and transports data. It doesn't do anything.
XHTML stands for eXtensible HyperText Markup Language. It is a XML-based version of HTML. It should be lowercases and nested.
HTML5 stands for Hypertext Markup Language 5. It is the latest version of HTML.
