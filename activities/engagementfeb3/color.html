2. What is the difference between an ID and a Class

ID is to identify the name. It's an attribute. It can be used only once. It should be unique, not duplicated. (use less) generally, we should put a noun to name

Class can be named more than 1 tag. 1 tag can have more than 1 class. generally, we should put an adjective to name

3. What is the difference between a SPAN tag and a DIV tag?

SPAN doesn't show any changes on the webpage. You can put class in it. You can put span tag in the sentence. Span tag doesn't cause the line break.

DIV tag causes the line break.

4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV tag is not a semantic and can include the class to name.

However, the others are semantic. They have their clear meaning for each tag.
