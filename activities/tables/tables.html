ENGAGEMENT

1. When do you use a table?

When you put data, schedule, etc.

2. When do you not use a table?

When you layout the elements

3. When do you need to do extra work for table accessibility?

Screen readers. Use "th" instead of "td."

"thead" is for the 1st row (labels)

"tbody" is for data

"tfooter" is averages, totals, summary of the data.

NEW ENGAGEMENT

1. Why don't we use the border attribute?

If we use the border attribute, there is no way to change the style like get rid of border or change the color.

And also, we would get a validation eroor.

2. What is the easiest way to color every other row a different color?

There is a function to selelct row automatically, ":nth-child(odd)." Or use a class name and apply colors in CSS.
